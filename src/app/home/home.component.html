List of bot commands


Command Description Usage
iline Shows IRCnet server I-lines !iline <none/nick/host>
server Shows linked servers !server <tld/sid/open>
spoof Shows servers available for cloaked connection (Spoof) !spoof <none>
ipv6 Shows IPv6 servers !ipv6 <tld/open>
ip Resolves IPv4/6 and shows whois !ip <ip/host>
ping6 Pinging IPv6 address !ping6 <ipv6/host>
splits Shows the current server splits (alert enabled on channel) !splits <none>
kills Shows the last 3 kills from IRCops (alert enabled on channel) !kills <none>
collide Shows the last 3 nick collisions !collide <nick>
seen Shows the last seen of a nick !seen <nick>
ircop Shows online IRCops !ircop <server>
uptime Shows server uptime !uptime <server>
info Shows channel modes and topic !info <chan>
reop Shows channel reop list !reop <chan>
invite Shows channel invite list !invite <chan>
ban Shows channel ban list !ban <chan>
